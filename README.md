🩺 Diabetes Prediction with AI

A Streamlit web app that predicts diabetes risk using a machine learning model, 
explains predictions with SHAP, and includes an AI Health Assistant powered by 
LangChain + FAISS + Groq (Llama 3).

Features:
- 🔬 ML-based diabetes risk prediction
- 💬 RAG chatbot for diabetes Q&A (LangChain + FAISS + Groq)
- 🧠 Natural language input — describe your health in plain English
- 📊 SHAP explainability for model predictions
- 📈 Model performance metrics
- 🚀 Deployed on Streamlit Community Cloud

Tech Stack:
Python · Streamlit · Scikit-learn · LangChain · FAISS · Groq (Llama 3) · SHAP · HuggingFace Embeddings

Dataset Link:https://www.kaggle.com/datasets/jamaltariqcheema/pima-indians-diabetes-dataset
